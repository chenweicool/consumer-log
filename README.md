# consumer-log
 怎么启动项目



#### 前端项目 log-manager  

1，进入log-manager目录中，执行 npm install依赖安装完成  

2， 执行npm run dev 就可启动前端项目，后端项目启动后，就可以使用前端登陆了。

   默认用户名和密码

  admin  123456        



#### 后端项目 log-server

1， 导入项目，下载依赖，

2，导入logsdata.sql这个文件到数据库，初始化数据库

3， 修改配置文件application.properties中的数据库的配置信息

4，  启动项目，就可以进行访问相应的接口内容了。
